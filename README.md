# 项目说明文档

## BattleControl
战斗系统的预留模块（可能会有）。

## CameraController
摄像头跟随模块，使用 Cinemachine 实现。

## Catch
娃娃机功能，目前处于测试阶段。

## Manager
仅列出几个主要类
### SuperControl
- **Data**：定义了数据类。
- **SuperController**：单例控制类，除了数据管理外，还实现了暂停等功能。
- **PlotManager**：单例控制类，主要服务于 Gal 的剧情展示。

### Static
用于存档的静态类。

## Restaurant
奶茶店玩法实现模块。

## StartMenu
起始页 UI 模块。

## SystemUI
测试中的其他 UI 模块。
